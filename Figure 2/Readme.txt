This folder contains the code to replicate Figure 2 in the article "Mixing times of data-augmentation Gibbs samplers for high-dimensional probit regression" by F.Ascolani and G.Zanella.

It includes the following files:

- "Functions_coupling.R", which contains the functions to implement all the couplings.

- "Simulations_supplementary.R", which contains the code to replicate Figure 2.



See Appendix E for the details of the simulation.

NOTE: all the files admits the possibility of generating imbalanced data (left plot of Figure 2) or data generated from the model (right plot of Figure 2). In the code this is highlighted by the comment "#generate the y: COMMENT THE ONE YOU DO NOT USE"
